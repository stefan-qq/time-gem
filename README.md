# Time Gem
De-clutter your mind, structure your day, FOSS style!
</div>

> [!WARNING]
> **Project Status:** Time Gem is currently in its **very early development stages**. Features are evolving rapidly! 
> 
> **Roadmap Ideas:** We are actively exploring options for **fully private, secure syncing between devices** and potentially developing a **desktop PC app** down the line.

## Description
Time Gem is a Free and Open Source (FOSS) note-taking, time-management, and overall life-management tool built for Android using Jetpack Compose. 

## Features
* **Modern Material 3 Interface:** Full support for Material You dynamic color themes and adaptive layouts.
* **Monochrome Icon Support:** Fits perfectly into clean, minimalist Android home screens.
* **FOSS & Privacy-First:** 100% open-source with absolutely zero tracking or data collecting.

## Requirements
* Android 8.0 (API 26) or above
